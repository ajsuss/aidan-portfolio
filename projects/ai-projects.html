AI Projects

Exploring search, multi‑agent systems, reinforcement learning and inference.

Search Algorithms

I implemented path‑finding agents for Pacman using depth‑first, breadth‑first, uniform‑cost and A* search. Custom heuristics were developed for complex multi‑goal problems.

Multi‑Agent Search

Adversarial search algorithms such as minimax, alpha‑beta pruning and expectimax were used to extend search to multi‑agent settings. Evaluation functions were tuned for ghost avoidance and food collection in the Pacman environment.

Reinforcement Learning

I implemented value iteration, Q‑learning and approximate Q‑learning to teach agents optimal behaviour across grid worlds, robot crawlers and Pacman environments.

Probabilistic Inference

Bayes net and Hidden Markov Model trackers were built for the “Ghostbusters” project. Techniques included variable elimination, particle filtering and belief‑state updates.

Machine Learning

I developed perceptron classifiers and feed‑forward and recurrent neural networks for tasks like digit recognition and language identification, practising hyper‑parameter tuning and training pipelines.
