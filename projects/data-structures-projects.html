Data Structures Projects

Implementing games and version control in Java.

Blocks

I developed a two‑player strategy game on an N×N grid featuring extend and jump moves with automatic captures. This project honed my Java control flow and array manipulation skills.

Enigma

I built a complete software simulator of the WWII Enigma machine. The simulator included classes for rotors, reflectors, plugboard and encryption logic, and implemented rotor stepping and reversible permutations.

Ataxx

I implemented the board game Ataxx on a 7×7 grid with a computer opponent using depth‑limited minimax with alpha‑beta pruning and a custom heuristic. A GUI and undo functionality were integrated.

Gitlet

I created a miniature Git‑style version control system supporting init, add, commit, branch, merge and log. The project used SHA‑1 hashes and efficient data structures to manage repository state.
