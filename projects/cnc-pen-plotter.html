CNC Pen Plotter

Custom XY‑gantry plotter with servo‑driven pen lifter and FluidNC firmware.

Project Overview

This project involved developing a CNC pen plotter that converts digital drawings into precise line art. The system features a custom pen‑retraction mechanism for rapid tool lifts and a modular control architecture.

Mechanical Design Highlights

The open‑frame XY gantry was built from extrusion rails and driven by dual stepper motors with DRV8825 microstepping. Limit switches enabled homing and a drag‑chain cable management system kept wiring neat. Compact 3D‑printed brackets and datum features ensured repeatable assembly.

Pen Lifter Mechanism

A servo‑driven slider converts rotary motion into vertical pen lift via a linkage. Idle servo torque and pen compression remove the need for a return spring. An adjustable mounting bracket with chamfered pegs, datum holes and threaded inserts ensures proper alignment, and a limit‑switch stopper enforces consistent travel.

Hardware & Integration

The plotter uses an ESP32 running FluidNC firmware with DRV8825 drivers. The pen lifter mounts on the Z‑axis carriage while the X and Y axes are driven by GT2 belts and T‑slot sliders.

Software & Firmware

FluidNC provides motion control. I developed a Python/Tkinter GUI for G‑code generation, real‑time plotting and commands such as E‑stop, home and reset.

Outcomes

The plotter achieved clean, disconnected strokes on both whiteboard and paper. Tool‑swap snap‑fit posts support multiple pen sizes, paving the way for future automatic tool changers. Real‑time feedback via MQTT ensures synchronous drawing.
